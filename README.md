# Stock-Price-Prediction-Model-STOCKSGROW-
StocksGrow is a web-based stock analysis and forecasting application built using Streamlit and Machine Learning. It provides a dashboard, detailed stock analysis, and future price prediction using an LSTM model. The system supports login/registration with MySQL and analyzes stocks, crypto, and commodities.
